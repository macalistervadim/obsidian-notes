# *args

`*args` принимает в качестве аргументов любое кол-во элементов (ПОЗИЦИОННЫХ!) и помещает их в кортеж с именем args

```python
def a(*args):
	return args

print(a("a", [1, 2, 3], ("a", 1))) # ('a', [1, 2, 3], ('a', 1))
```

>[!info] о имени args
>ясное дело, что после знака `*` мы можем хоть написать `aleksey`, но я думаю что другие разработчики явно не будут рады такой шутке - лучше предерживаться PEP8, в котором явно указано, что для прочих позиционных аргументов (*) следует использовать имя `args`


# kwargs

`**kwargs`  также работает как и `*args` но при этом помещая ИМЕНОВАННЫЕ аргументы в словарь, где ключ - это имя аргумента, значение - значение этого аргумента

```python
def a(*args, **kwargs):
	return args, kwargs


print(a("a", [1, 2, 3], ("a", 1), a={1: 2}, n=[1, 2, 3])) # (('a', [1, 2, 3], ('a', 1)), {'a': {1: 2}, 'n': [1, 2, 3]})
```

точно такое же правило о названии как и в `*args` действует на `**kwargs`
